# Luxe Scents — Static Site

This is a small static site for the "Luxe Scents" perfume brand.

Files included:
- `index.html` — main site
- `styles.css` — styles
- `script.js` — simple JS for cart & form
- `README.md` — this file

How to preview locally:
- Option A: open `index.html` in a browser (some browsers restrict JS for local files)
- Option B: run a simple HTTP server:
  - Python 3: `python -m http.server 8000`
  - Node (npm): `npx http-server . -p 8000`
  - Then open http://localhost:8000

Deploy to GitHub Pages:
- I pushed these files to the `gh-pages` branch. GitHub Pages will publish the site at: https://sakinabanuziana.github.io/dr-physio-site/
